# Route53 Module

Manages AWS Route53 hosted zones, DNS records, and health checks. Supports both creating new hosted zones and adding records to existing ones, with optional alias records for AWS resources (ALB, CloudFront, etc.) and configurable health checks.

Naming convention: `${var.project}-${var.environment}-{service}-{resource}`
